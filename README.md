#  AI Chat Assistant

This is a chat-based AI web application built using Flask.

##Features
- Chat-style interactive UI
- Real-time response generation
- Clean and modern design

## Tech Stack
- Python
- Flask
- HTML & CSS

## About the Project
This is the initial version of the project where responses are logic-based.

## Future Improvements
- Integration of real AI APIs (Generative AI)
- Chat history storage using database
- More intelligent responses

pip install flask
